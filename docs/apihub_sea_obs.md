# KMA API Hub - 해양기상관측자료 조회

기상청 API Hub(apihub.kma.go.kr)에서 제공하는 해양기상관측 API 명세.

해양기상부이, 파고부이, 표류부이에서 관측한 파고, 파주기, 파향, 수온 등의 데이터를 조회할 수 있다.

---

## 인프라 현황

- **해양기상부이**: 17개 관측소 (1996년 7월~)
- **파고부이**: 59개 관측소 (2009년 1월~)
- **생산 주기**: 매시 :30, :00분 + 일별 요약

---

## API 1: 해양기상관측 종합 (sea_obs.php)

**Endpoint:**

```
https://apihub.kma.go.kr/api/typ01/url/sea_obs.php
```

**설명:** 해양기상부이, 파고부이, 표류부이에서 관측한 파고, 파주기, 파향, 수온 등의 종합 관측 데이터를 제공한다.

### 요청 파라미터

| 파라미터 | 설명 |
|----------|------|
| tm | 날짜/시각 (KST, YYYYMMDDHHmm). 기본값: 현재시각 |
| stn | 관측소 번호 (0=전체) |
| help | 도움말 플래그 (1=필드 설명 포함) |
| authKey | API 인증키 |

### 응답 필드

| 필드 | 설명 |
|------|------|
| TP | 관측 유형 |
| STN_ID | 관측소 ID |
| STN_KO | 관측소명 |
| TM | 관측 시각 |
| WH | 유의파고 (m) |
| WD | 풍향 (deg) |
| WS | 풍속 (m/s) |
| WS_GST | 돌풍풍속 (m/s) |
| TW | 해수면 수온 (℃) |
| TA | 기온 (℃) |
| PA | 해면기압 (hPa) |
| HM | 상대습도 (%) |

### 응답 형식

- Content-Type: `text/plain`
- `#`로 시작하는 줄은 헤더/메타 정보
- 데이터는 공백으로 구분된 고정폭 텍스트
- 결측값: `-9`, `-99`

### 요청 예시

```
https://apihub.kma.go.kr/api/typ01/url/sea_obs.php?tm=202604160000&stn=0&help=1&authKey={인증키}
```

## API 2: 해양기상관측 기간 조회 (kma_buoy2.php)

**Endpoint:**

```
https://apihub.kma.go.kr/api/typ01/url/kma_buoy2.php
```

**설명:** 특정 기간의 해양기상관측 데이터를 조회한다. 시작/종료 시각을 지정하여 범위 조회가 가능하다.

### 요청 파라미터

| 파라미터 | 설명 |
|----------|------|
| tm1 | 조회 시작 (YYYYMMDDHHmm KST) |
| tm2 | 조회 종료 (YYYYMMDDHHmm KST) |
| stn | 관측소 번호 (0=전체) |
| help | 도움말 플래그 (1=필드 설명 포함) |
| authKey | API 인증키 |

### 응답 형식

sea_obs.php와 동일한 고정폭 텍스트 형식으로 응답한다.

### 요청 예시

```
https://apihub.kma.go.kr/api/typ01/url/kma_buoy2.php?tm1=202604150000&tm2=202604160000&stn=0&help=1&authKey={인증키}
```

## API 3: 해양기상부이 상세 (kma_buoy.php)

**Endpoint:**

```
https://apihub.kma.go.kr/api/typ01/url/kma_buoy.php
```

**설명:** 해양기상부이에서 관측한 상세 데이터를 제공한다. 풍향/풍속 센서 이중화, 파고 상세 통계 등 sea_obs.php보다 세분화된 필드를 포함한다.

### 요청 파라미터

sea_obs.php와 동일 (tm, stn, help, authKey).

### 응답 필드

| 필드 | 설명 |
|------|------|
| WD1 | 풍향 - 센서 1 (deg) |
| WD2 | 풍향 - 센서 2 (deg) |
| WS1 | 풍속 - 센서 1 (m/s) |
| WS2 | 풍속 - 센서 2 (m/s) |
| WS1_GST | 돌풍풍속 - 센서 1 (m/s) |
| WS2_GST | 돌풍풍속 - 센서 2 (m/s) |
| PA | 해면기압 (hPa) |
| HM | 상대습도 (%) |
| TA | 기온 (℃) |
| TW | 수온 (℃) |
| WH_MAX | 최대파고 (m) |
| WH_SIG | 유의파고 (m) |
| WH_AVE | 평균파고 (m) |
| WP | 파주기 (sec) |
| WO | 파향 (deg) |

### 응답 형식

sea_obs.php와 동일한 고정폭 텍스트 형식으로 응답한다.

## 참고 사항

- 모든 시각 파라미터는 KST(한국표준시, UTC+09:00) 기준이다.
- 인증키는 기상청 API Hub(https://apihub.kma.go.kr)에서 발급받는다.
- 환경변수 `KMA_AUTH_KEY`에 인증키를 설정하여 사용한다.
- 이 프로젝트에서는 `packages/collect`의 KMA Sea 클라이언트가 해당 API를 호출한다.
