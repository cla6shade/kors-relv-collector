# scripts

Python utility scripts. Run via the project's `.venv`.

## Setup

```bash
python3 -m venv .venv
.venv/bin/pip install -r requirements.txt
```

## convert_hwp_docs.py

Converts `docs/*.hwp` API specs to AI-readable Markdown (`docs/*.md`).

```bash
.venv/bin/python scripts/convert_hwp_docs.py
```
